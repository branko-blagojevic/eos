Install EOS via yum
-------------------

For client

.. code-block:: text

   yum install eos-client

For server

.. code-block:: text

   yum install eos-server eos-client eos-testkeytab eos-fuse jemalloc

For HTTP server

.. code-block:: text

   yum install eos-nginx

For GridFTP server

.. code-block:: text

   yum install xrootd-dsi

For EOS test on MGM (namespace node)

.. code-block:: text

   yum install eos-test
